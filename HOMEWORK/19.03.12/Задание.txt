Взаимодействие с пользователем

1) Создайте приветствие. prompt с вопросом как вас зовут и alert с текстом например: "Привет, Алексей";
2) Создайте калькулятор с выбором знака.
prompt для первого числа
prompt для второго числа
prompt для знака (+ - / *)

Внутри скрипта вы проверяете какой знак ввел пользователь и на основе этой информации вычисляете значение.
